蘇恩手操攻略整理

整理方向：手操連段效益、技能選擇、畢業因子、加護、界限突破，以及 AI 使用評價。內容依 Maygi 蘇恩指南、Relink Wiki 技能資料、GameFAQs 專屬因子表與傷害計算機資料整理。

角色：蘇恩 / Tweyen 定位：遠程蓄力、多重鎖定、弱體爆發 推薦：手操高於 AI

快速結論

蘇恩的核心不是亂射，而是「技能疊 Ultrasight → 滿層多重鎖定 → 空窗依距離選連段」。熟練後打大體型王很舒服，AI 則比較難發揮完整價值。
項目 重點
主輸出 滿 Ultrasight 的多重鎖定射擊
次輸出 蓄力連段終結、致命一擊、殺戮之雨、散環
主要資源 Ultrasight 等級
操作重點 用技能快速堆 Ultrasight，滿層後多重鎖定，空窗用連段補傷害與資源
適合目標 大體型、動作不太亂的王，例如貝西摩斯

連段效益

以下用 X 表示普攻，Y 表示終結。Maygi 指南的重點是：第一段普攻蓄力效率較差，後面段數與終結比較值得蓄力，終結盡量蓄滿。

連段 用途 實戰判斷
XYY 位移、補一點 Ultrasight 不是主 DPS，用來調位置或差一點滿層時補資源
XXYY 近距離輸出 貼近王且箭雨命中率高時，輸出很好
XXXYY 泛用主力 距離較遠、王會動、想穩定累積 Ultrasight 時優先 
近距離能吃滿箭：XXYY
距離不穩或王會動：XXXYY
只要移動或補一點資源：XYY

多重鎖定、終結、致命一擊後可以用閃避取消，主要價值是縮短硬直、修正站位，讓下一輪連段或技能更快接上。

推薦技能

技能 推薦度 用途
絕騎 / Peerless Eye 必帶 讓 Ultrasight 累積效率大幅提高，是爆發核心
腐化 / Depravity 必帶 快速疊 Ultrasight，附多種弱體與 DOT
致命一擊 / Clincher 必帶 高傷害技能，敵人弱體越多越痛
殺戮之雨 / Lethal Rain 泛用第四格 傷害與暈眩都好，穩定泛用

共鳴虹光 / Merculight 防禦向 攻擊上升與鏡像，輸出向通常不優先

基本循環

開場：絕騎
→ 腐化 / 致命一擊 / 第四技能堆 Ultrasight
→ Ultrasight 滿後打多重鎖定
→ 空窗用 XXYY 或 XXXYY
→ 技能好了再堆 Ultrasight
→ 重複

Link Time 思路

多重鎖定
→ 閃避取消
→ 技能或連段補 Ultrasight
→ 再多重鎖定

實戰上不必死背固定迴圈，重點是保持「滿層多重鎖定」的頻率，並用距離合適的連段填空窗。

專屬因子

戰氣 / Dark Huntress's Warpath 蓄力變快，蓄力攻擊強化傷害 +20%，暈眩提升，機率上弱體 手操強烈推薦

畢業因子方向

類型 推薦 理由
上限 傷害上限 65，之後看 Alpha / Beta / Gamma 先讓主輸出有天花板，再依計算機補路西法因子
基礎輸出 暴君、蓄力攻擊、技能傷害、渾身或捨身 蘇恩很多傷害來源需要先推到上限
蘇恩特別重要 技能加速、怒濤、快速蓄力 提高技能循環、資源累積與蓄力手感
追加傷害 追擊、有利屬性轉換、狂戰士追擊 追擊與有利屬性很強；狂戰士追擊要看面板攻擊是否能觸發
功能 明鏡止水、藥水持有、自動復活、暈眩 高難與刷關穩定度用
可選 連技終結 如果你的終結段沒碰上限，可作為補傷害選項

Maygi 指南特別提到：蘇恩 DOT 有一定占比，而 DOT 比較看面板攻擊，所以 暴君很有價值。如果傷害計算機顯示沒碰上限，先補實際傷害；碰上限後再補 Alpha / Beta / Gamma。

界限突破

優先 詞條 理由
1 普通攻擊傷害上限 連段與多重鎖定相關輸出吃得到
2 攻擊力 +1000 蘇恩 DOT 也吃面板攻擊，很有價值
3 爆擊率 如果加護或因子已補滿，可降低優先
4 技能傷害上限 致命一擊、殺戮之雨、散環等技能吃得到

AI 評價

蘇恩是 手操價值遠高於 AI 的角色。AI 不會像玩家一樣穩定管理 Ultrasight、距離、蓄力、多重鎖定與閃避取消，因此很難打出手操上限。

用途 評價
手操刷貝西摩斯 可練，遠程安全，大體型王相性佳
手操高難 可用，但需要熟悉蓄力、距離與技能節奏
AI 隊友 不太推薦當主力，穩定性與輸出都不如頂級 AI
